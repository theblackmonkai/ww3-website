# WW3 Information Portal

A premium Apple-style website about World War III, featuring:

- 🎨 Modern, minimalist design inspired by Apple's product pages
- 📱 Fully responsive layout for all devices
- ✨ Smooth animations and parallax effects
- 🌙 Dark mode with glassmorphism effects
- 📊 Interactive timeline and data visualizations

## Features

- **Hero Section**: Dramatic introduction with animated gradient orbs
- **Timeline**: Scroll-triggered animations showing key events
- **Players Section**: Cards for major nations and coalitions
- **Impact Section**: Visual representation of global effects
- **Analysis**: Expert commentary with key statistics

## Tech Stack

- Pure HTML5, CSS3, JavaScript
- No frameworks or dependencies
- Responsive design with CSS Grid and Flexbox
- Intersection Observer API for scroll animations

## Deployment

### Vercel (Auto-deploy)

This repository is connected to Vercel and will auto-deploy on every push to main.

### Manual Deployment

```bash
# Deploy to Vercel
vercel --prod

# Or deploy to Netlify
netlify deploy --prod
```

## Local Development

```bash
# Serve locally
npx serve .

# Or with Python
python -m http.server 8000
```
